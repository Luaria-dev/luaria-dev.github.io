# Sales Dashboard (ECharts)

Static, GitHub Pages-friendly dashboard powered by **ECharts**.

## Files
- `index.html` — dashboard UI + charts
- `data.json` — exported data from Excel (Base de Datos sheet)

## Run locally
```bash
python -m http.server 8000
# then open http://localhost:8000
```

## Publish (GitHub Pages)
1. Create a repo and upload `index.html` and `data.json`
2. Repo Settings → Pages → deploy from branch (root)
3. Open the provided Pages URL
